Export to CSV

Download all your questions as a CSV file. Open it in Google Sheets, then publish it to the web to use with your Quiz App. The columns will match exactly: ExamType, Domain, Subdomain, Question, OptionA, OptionB, OptionC, OptionD, CorrectAnswer, Rationale.

How to use with the Quiz App

1. Click Download CSV above
2. Go to sheets.google.com → File → Import → Upload the CSV
3. Go to File → Share → Publish to web → Sheet1 → CSV → Publish
4. Copy the URL and paste it into your Quiz App
