📋 How to set up your Google Sheet

  1. Open Google Sheets and create a new sheet
  2. Add these 10 column headers in Row 1 exactly as shown:
    ExamType Domain Subdomain Question OptionA OptionB OptionC OptionD CorrectAnswer Rationale
  3. For ExamType column, type either CP or SCP for each question
  4. For CorrectAnswer column, just type A, B, C, or D
  5. Go to File → Share → Publish to web → choose Sheet1 + CSV → Publish
  6. Copy the URL and paste it above
